1. **Install Rust**

2. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/raycasting-tron.git
   ```

3. **Download dependencies**  
   ```bash
   cargo build
   ```

4. **Run the game**  
   ```bash
   cargo run
   ```

## ğŸ® **Controls**

- **W/S**: Move forward/backward
- **A/D**: Turn left/right
- **Mouse**: Look around
